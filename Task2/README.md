Credit Card Fraud Detection

Project Description

This project detects fraudulent credit card transactions using machine learning algorithms.

Objective

To identify fraud transactions accurately while handling imbalanced data.

Approach

Data cleaning and preprocessing

Handling class imbalance

Training machine learning models

Evaluating performance using precision, recall, and ROC-AUC

Technologies Used

Python

Pandas, NumPy

Scikit-learn

Matplotlib, Seaborn

Output

The model classifies transactions as Fraud or Not Fraud.
