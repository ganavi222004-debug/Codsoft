3️⃣ Projects Included
Movie Genre Classification

Explanation:
This project predicts the genre of a movie based on its description or plot.
It uses Natural Language Processing (NLP) techniques such as text cleaning and vectorization along with machine learning classifiers.

Credit Card Fraud Detection

Explanation:
This project identifies whether a credit card transaction is fraudulent or genuine.
It focuses on handling imbalanced data and uses evaluation metrics like accuracy, precision, recall, and ROC-AUC to measure performance.

Customer Churn Prediction

Explanation:
This project predicts whether a customer is likely to leave a company or service.
It helps businesses understand customer behavior and take steps to reduce churn.

Spam SMS Detection

Explanation:
This project classifies SMS messages as Spam or Ham (Not Spam).
It applies NLP techniques such as text preprocessing and feature extraction to build an accurate classification model.
