Customer Churn Prediction

Project Description

This project predicts whether a customer is likely to leave a company or service based on customer data.

Objective

To help businesses reduce customer loss by predicting churn in advance.

Approach

Data preprocessing and feature engineering

Training classification models

Evaluating performance using accuracy, precision, and recall

Technologies Used

Python

Pandas, NumPy

Scikit-learn

Data Visualization tools

Output

The model predicts Churn or No Churn for each customer.
