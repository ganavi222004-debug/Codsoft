Movie Genre Classification

Project Description

This project predicts the genre of a movie based on its plot or description using Machine Learning and Natural Language Processing (NLP) techniques.

Objective

To automatically classify movies into different genres using text-based features.

Approach

Text preprocessing (lowercasing, stopword removal)

Feature extraction using NLP techniques

Training machine learning classification models

Evaluating model performance using accuracy

Technologies Used

Python

Pandas, NumPy

Scikit-learn

NLP (TF-IDF / Count Vectorizer)

Output

The model predicts the most suitable movie genre for a given movie description.
