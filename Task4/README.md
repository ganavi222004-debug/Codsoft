Spam SMS Detection

Project Description

This project classifies SMS messages as Spam or Ham (Not Spam) using machine learning and NLP.

Objective

To automatically detect spam messages and improve communication safety.

Approach

Text preprocessing and cleaning

Feature extraction using NLP

Training machine learning classification models

Model evaluation using accuracy and confusion matrix

Technologies Used

Python

Pandas, NumPy

Scikit-learn

NLP (TF-IDF / Count Vectorizer)

Output

The model classifies messages as Spam or Not Spam.
